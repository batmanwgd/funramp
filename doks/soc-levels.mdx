## System and Organizational Control (SOC)

_When we talk about SOC, we are normally talking about the reports that result from a SOC audit, so another way you could think of this topic area would be SOCR_

| Concept             | Description                                                                                                                        |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| Purpose            | Provides auditor to auditor communication on the implementation of controls at the service organization based on “trust criteria”. |
| Tiers              | SOC1, SOC2, SOC3                                                                                                                   |
| Authorization Group          | American Institute of Certified Public Accountants (AICPA)                                                                         |
| Reporting Standard | SSAE 18 issued by AICPA and used for crosstalk between Federal and Commercial Industries                                           |
| Key Terms | <ul><li>Control Objectives & Activities</li><li>Service Auditor</li><li>Service Organization</li><li>SSAE 18</li><li>Subservice Organization</li></ul> |
